# Titan API Server

## Requirements

Requires `node ^10.15.0`

## Commands

Here are the commands to do common actions:

- Installation: `yarn`
- Start server with auto-restart on change: `yarn start:dev`
- Linting: `yarn lint`
- Compiling TS to JS -> dist: `yarn build`
